Udacity Arcade Game

In this project that was completed as a part of Udacity's Frontend Nanodegree track, I built a "Frogger" inspired arcade game completely in vanilla Javascript. I was provided with a game looping engine, which I customized according to the direction I wanted to go, an image caching utility, and visual game assets. I used my newfound knowledge of the principles of Object Oriented Programming with Javascript, pseudoclasses, prototypes, and object inheritance to build character classes and subclasses, define their properties(starting locations, etc.) and their behavior(movement speed, player controls, etc.).
