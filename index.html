Calorie Tracking Application

In my latest project utilizing React and Redux, I created this single page application that allows users to log and track their daily caloric intake. It utilizes Firebase for user authentication and a database to store profile and user entries. It also integrates the Nutritionix API so that users can search for and retrieve food items and their associated calorie count/nutritional data, and add those to their daily entries. This application also informs the user of their daily caloric intake target based on their weight management goal and profile data, and also alerts the user if their average daily intake is meeting their daily target or not.
